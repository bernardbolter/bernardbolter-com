06
Spacing Scale

Base unit: 4px. Tailwind default scale — all spacing in multiples of 4.

1 / 4pxMicro — icon gaps, tight inline
2 / 8pxXS — between label and heading
3 / 12pxSM — card internal padding
4 / 16pxMD — between elements in a card
6 / 24pxLG — section internal gap
8 / 32pxXL — between cards in a grid
12 / 48px2XL — section top/bottom padding
16 / 64px3XL — hero padding, page margins
24 / 96px4XL — between major page sections
