Section 4 — Z-Index

Z-Index Layers

Seven named layers. Never use arbitrary values. Pick a layer by purpose, then use +1 for stacking within the same group.

0 z-base Normal document flow
1000 z-artwork Artwork images, timeline content
2000 z-chrome Info panel, name container, info content
3000 z-nav Title box, magnify button, artwork image controls
4000 z-overlay Slideshow, filter drawer, search drawer, switcher
5000 z-ui-top Close buttons, nav buttons
9000 z-modal Lightbox, Klaro cookie modal

Note: Artwork title in slideshow mode sits at z-modal - 1 (8999) — above overlay but below true modals.

Section 4 — Breakpoints

Breakpoints

Four custom breakpoints. Never use Tailwind defaults (sm, md, lg). Always use s, m, l, xl.

s:550px
m:768px
l:979px
xl:1200px

Grid switches: 1 col (default) → 2 col (s) → 3 col (m) → 4 col (l) → 5 col (xl). Max grid width: 1500px.

Section 6

Artwork Sizing — Size Tiers

Artworks display at sizes that represent their real-world physical scale. Always use useArtworkDimensions hook. Never normalize to uniform grid cells. Never use object-cover.

Visual impression of size tiers — landscape orientation, same container. The larger the real artwork, the larger it appears on screen.

sm · factor 0.65
small works
md · factor 0.75
medium
lg · factor 0.85
large (default)
xl · factor 0.95
large-scale works

Square artworks use reduced factors (xl 0.90, lg 0.80, md 0.70, sm 0.60). Portrait artworks are height-constrained. Landscape artworks are width-constrained.

Section 8

Button SVG Language

Same 3D box principle as the title box. Shadow shape stroke = #999 (lighter, recedes). Icon fill = #666 (darker, comes forward). Shell never changes — only the icon swaps.
